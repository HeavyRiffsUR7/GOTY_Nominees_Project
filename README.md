## Dataset Features

The dataset includes nominee-level information such as:

- **Game Title**
- **Year of Nomination**
- **Award Category**
- **Publisher / Studio**
- **Platforms**
- **Genre Tags**
- **Critic vs User Reception Signals** *(optional expansion)*

This structure allows deeper insights into what types of games tend to dominate award recognition.

---

## Analysis Goals

This project aims to answer questions such as:

- Which studios or publishers appear most frequently in GOTY nominations?  
- Are certain genres consistently overrepresented?  
- How do release timing and platform availability affect nomination frequency?  
- What patterns emerge when comparing critic scores vs player sentiment?
